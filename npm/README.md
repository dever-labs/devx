# devx npm package

This package installs the [devx](https://github.com/dever-labs/dever) binary for your platform.

## Install

```sh
npm install -g @dever-labs/devx
devx version
```

## Supported platforms

| OS      | amd64 | arm64 |
|---------|-------|-------|
| Linux   | ✅    | ✅    |
| macOS   | ✅    | ✅    |
| Windows | ✅    | ✅    |
